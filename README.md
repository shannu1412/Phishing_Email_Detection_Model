# PhishGuard - Phishing Email Detection System

## Overview

PhishGuard is a Machine Learning-based cybersecurity project that detects phishing emails by analyzing email content and identifying suspicious patterns. The system classifies emails as **Phishing** or **Safe** using Natural Language Processing (NLP) and machine learning techniques.

## Features

* Email text preprocessing and cleaning
* URL detection and analysis
* TF-IDF feature extraction
* Phishing email classification
* Accuracy evaluation
* Confusion Matrix visualization
* Real-time email prediction

## Technologies Used

* Python
* Scikit-learn
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Project Workflow

1. Load and preprocess email dataset
2. Extract textual features using TF-IDF
3. Train machine learning model
4. Classify emails as Phishing or Safe
5. Evaluate model performance using accuracy and confusion matrix

## Machine Learning Model

* Logistic Regression

## Dataset

The dataset contains phishing and legitimate email samples with corresponding labels:

* phishing
* safe

## Results

* Achieved high classification accuracy on test data
* Successfully identified phishing emails based on textual content and URL patterns
* Visualized performance using a confusion matrix
